# Snake Game

A simple browser-based Snake Game built with JavaScript, HTML, and CSS.

## Overview

This project is a classic Snake Game where the player controls a snake, collects food, and tries to achieve the highest score possible without colliding with walls or itself.

## Features

- Classic snake movement
- Score tracking
- Increasing challenge as the game progresses
- Built with plain JavaScript, HTML, and CSS
- Lightweight and easy to run locally

## Tech Stack

- JavaScript
- HTML
- CSS

## Getting Started

### Prerequisites

You only need a modern web browser to run this project.

### Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/frankaiken/snake-game.git
   ```

2. Open the project folder:

   ```bash
   cd snake-game
   ```

3. Open the `index.html` file in your browser.

## How to Play

- Use the arrow keys to control the snake
- Eat the food to grow longer and increase your score
- Avoid hitting the walls or the snake’s own body
- Try to get the highest score you can
